PUB/LibreCat docker image

mongoDB @: https://github.com/docker-library/mongo/tree/4bb17b336a05ad85c9bf83b103d21529e27e62f9/3.2

LibreCat @: https://github.com/LibreCat/LibreCat

'docker-compose' instruction.
